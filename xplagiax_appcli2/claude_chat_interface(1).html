Usuario: QUIERO UN CODIGO EN PYTHON PARA USAR EN UNA RASPBERRY PI 3 EL GPIO CON 4 SERVOS QUE AL ACTIVARSE GIREN EN 90 GRADOS

Claude: Te ayudo a crear un código en Python para controlar 4 servos en una Raspberry Pi 3. Aquí tienes el código completo con todas las características necesarias.


Características:

  • Control de 4 servos mediante GPIO
  • Rotación de 90 grados
  • Frecuencia PWM de 50Hz
  • Menú interactivo
💻 Código Python
python 
import RPi.GPIO as GPIO
        import time

        # Configuración de GPIO
        GPIO.setmode(GPIO.BCM)
        GPIO.setwarnings(False)

        # Pines GPIO para los servos
        servo_pins = [18, 19, 20, 21]

        # Configurar los pines como salida
        for pin in servo_pins:
            GPIO.setup(pin, GPIO.OUT)

        # Crear objetos PWM para cada servo
        servos = []
        for pin in servo_pins:
            servo = GPIO.PWM(pin, 50)
            servo.start(0)
            servos.append(servo)

        def angle_to_duty_cycle(angle):
            """Convierte un ángulo (0-180°) a duty cycle"""
            return 2.5 + (angle / 180.0) * 10.0

        def move_servo(servo_index, angle):
            """Mueve un servo a un ángulo determinado"""
            if 0 <= servo_index < len(servos) and 0 <= angle <= 180:
                duty_cycle = angle_to_duty_cycle(angle)
                servos[servo_index].ChangeDutyCycle(duty_cycle)
                print(f"Servo {servo_index + 1} movido a {angle}°")

        def rotate_all_90_degrees():
            """Gira todos los servos 90 grados"""
            print("Moviendo a posición inicial (0°)...")
            for i in range(len(servos)):
                move_servo(i, 0)
            time.sleep(2)
            
            print("Girando todos los servos 90°...")
            for i in range(len(servos)):
                move_servo(i, 90)
            time.sleep(2)

        # Programa principal
        if __name__ == "__main__":
            try:
                rotate_all_90_degrees()
            except KeyboardInterrupt:
                print("\\nInterrumpido")
            finally:
                for servo in servos:
                    servo.stop()
                GPIO.cleanup()
